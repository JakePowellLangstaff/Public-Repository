# Assembly Endless Runner (x86_64 / 68000 Conversion Project)

**Author:** Jake Powell
**Created:** December 2023
**Purpose:** Conversion of 68000 Assembly to x86_64 Assembly

---

## Overview

This project is a simple endless runner game written in assembly language.

It was originally built for the 68000 architecture and later modified as part of a conversion project. A health bar system was also added during development.

---

## How to Run

The main file you need is the `.asm` file.

It is designed to be run using **Easy68k**.

---

## Project Structure

All project files are located in:

```
final_project_1_c00287662
```

## Notes

* This is an educational project focused on assembly language and low-level programming.
* The game logic is very simple and was mainly used to explore architecture conversion and memory handling.
