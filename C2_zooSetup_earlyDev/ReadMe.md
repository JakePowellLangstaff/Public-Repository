This project explores the design and analysis of a simulated Command and Control (C2) framework for 
educational purposes within a controlled laboratory environment. 
The primary aim is to understand how modern malware establishes reconnaissance, persistence, 
and communication channels, while simultaneously identifying effective defensive countermeasures.
